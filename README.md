Feedstock for [cuttools](https://www.ugr.es/~pittau/CutTools/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

CutTools: computing 1-loop amplitudes at the integrand level

## Architectures

State per architecture: see the platforms badge above or [anaconda.org/hep-forge/cuttools](https://anaconda.org/hep-forge/cuttools).


## Install

```bash
conda install -c hep-forge -c conda-forge cuttools
```
